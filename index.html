Dictionary for Developers

AJAX

Short for Asynchronous JavaScript and XML.

Bit

A bit (short for binary digit) is the smallest unit of measurement in computing. 8 bits make up 1 byte.

Browser

A software program that allows you to surf the web. Popular web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge and Internet Explorer.

Cache

When you download (read) a web page, the data is "cached," meaning it is temporarily stored on your computer. The next time you want that page, instead of requesting the file from the web server, your web browser just accesses it from the cache, so the page loads quickly. The downside to this is that if the cached web page is often updated, you may miss the latest version. If you suspect that the web page you're seeing is not the latest version, use the "refresh" button on your browser.

dev

Short for "developer".

developer

a person or company that creates new products, especially computer products such as software.

FTP

File transfer protocol (FTP) is a common method of transferring files via the internet from one host to another host.

JPEG/h3>

JPEG stands for Joint Photographic Experts Group, which was the committee that created the file format known as JPEG. The format is commonlyl used for photos displayed on the world wide web.

software

Software is a general term for the various kinds of programs used to operate computers and related devices. (The term hardware describes the physical aspects of computers and related devices.

LAN

A local area network (LAN) is a system that connects computers and other devices that share a common communications line and wireless link, generally within a limited geographical area such as a home or office building.

Linux

Linux is a type of operating system that is open source and is used by many developers.

Operating System

An operating system is the software that helps us accessing the resources and applications and manages resources. Examples of operating systems are Ubuntu, Windows 10, Macintosh, Debian, and Manjaro.

Open Source

Software for which the original source code is made freely available and may be redistributed and modified

Source Code

Source code is the collection of files containing instructions in a programming language that make up a program. Source code is commonly compiled into a program, or executable. It may also be run by an interpreter if it is an interpreted language such as Python or Ruby.

Programming Language

A programming language is a type of written language that is used to provide instructions to the computer. Examples of programming languages include C, C#, C++, Java, Ruby, Python, and Go.

IDE

Integrated Development Environment - an application that developers use for software development. IDEs will provide features such as a source code editor, debugger and intelligent code completion.

algorithm

A sequence of instructions or a set of rules that are followed to solve a task.

Java

a general-purpose object-oriented programming language

object-oriented

a type of programming organized around objects

object

a value stored in memory that referenced by an identifier
